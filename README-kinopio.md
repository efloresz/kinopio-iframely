# Kinopio iFramely

Heroku self-hosted version of iframely

## Setup

- clone repo
- `npm install`
- `git remote add iframely https://github.com/itteco/iframely.git`

## Run

- `node server`
- http://localhost:8061/iframely?url=https://www.are.na/block/5177236

## To update 

- `git pull iframely master`
